Space Lich Omega 2

Web Game
AngularJS, Node.js, Firebase, HTML5, CSS3

Space Lich Omega 2 is an incremental RPG that was inspired by the game Candy Box. The player goes through many levels collecting items and defeating enemies. The project was released on February 1st, 2016. Since then, ~30,000 players have attempted to defeat The Lich. It is a sequel to a Space Lich Omega which I developed in 2014. The original has had over 40,000 players to date.

The project was built with AngularJS and is running on a small Node.js server. I am also using Firebase to track player statistics.

Project Leek

2D Action Platformer
GDScript, C#

'Project Leek' is the development name for my current project. It is an action platformer with a planned release in early 2018. A prototype was developed in C# and now the full project is currently in development using the Godot Engine and GDScript, a python based language.

This is my most ambitious personal project to date and has been a lot of fun to work on so far!
