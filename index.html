League Stats

Web App
AngularJS, HTML5, CSS3, Bootstrap

League Stats allows the user to search a players match history for the game League of Legends using the RIOT Games API. I had to get a bunch of data from the API and then organize it repeatedly in a similar fashion so I used Angular to help.

The user can view a players past 10 game match history and their averages. They can also see the players current rank in ranked games, as well as the champions, and their respective stats, used in those games.

Space Lich Omega

Browser RPG
Javascript, jQuery, HTML5, CSS3

Space Lich Omega is an incremental RPG that was inspired by the game Candy Box. The player goes through many levels collecting items and killing enemies. When I first started learning, knowing someday I could make a game like this was a huge motivator. To date over 15,000 people have played. Seeing people post it themselves on websites in many languages was a humbling experience.

This was also my first project of this scale. I quickly learned having a consistent development approach was key to saving time. I ended up restructing everything into an object oriented approach.

Imgur Gallery

Web App
AngularJS, HTML5, CSS3, Bootstrap

The motivation for Imgur Gallery came from wanting to easily view all the images from a subreddit. I also used this as an oppurtunity to learn Angular. I used Yeoman to scaffold the app and also used the project to familiarize myself with Grunt and Bower.

When the user selects the subreddit they want to search a call to the Imgur API is made which brings back the images, which are then displayed using ng-repeat.
